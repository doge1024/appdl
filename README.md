# appdl
    
解决mac上不能下载pp助手上的越狱App

>在Mac上没有下载越狱App的地方，加上iOS11以后砸壳也不容易了,所以就使用flask+react写了一个单页服务    

问题

1. 如何运行？

      ```
#1. 进入dl-flask
cd dl-flask     
#2. 在python3环境下，执行
pip install -r requirements.txt   
python manage.py runserver
#3. 在浏览器进入 http://localhost:5000
      ```
